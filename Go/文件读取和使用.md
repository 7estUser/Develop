## 判断文件/文件夹是否存在
```go
_,err := os.Stat("./test")
//err 为 nil 表示文件/文件夹存在
if err == nil{
    fmt.Println("文件/文件夹存在")
}
//os.IsNotExist(err)判断为true,说明文件或文件夹不存在
if os.IsNotExist(err){
    fmt.Println("文件/文件夹不存在")
}
//如果返回的错误为其它类型,则不确定是否在存在
```
## 读取目录
```go
//读取目录
dir_list, err := ioutil.ReadDir("../plugin/")
if err != nil {
        return
    }
//遍历目录中的file
for i, v := range dir_list {
    //文件名称
    fmt.Println(i, "=", v.Name())
    //系统信息  Linux:*syscall.Stat_t
    fmt.Println(reflect.TypeOf(v.Sys()))
}
```
##  读取文件
```go
//读取整个文件
ioutil.ReadFile(filepath)
//打开文件，没有读取等
file, err := os.Open("c:/tmp.txt")
//最后关闭文件读取io流
defer file.close
//按行读取文件内容
scanner := bufio.Newcanner(file)
for scanner.Scan(){
    println(scanner.Text())
}
```
